### Connect Repo to GitHub

> Replace `<your-username>` with your GitHub username:

```bash

```

---

### Push Local Repo to GitHub

```bash
git branch -M main

git push -u origin main
```

---

### Clone a Repo from GitHub

```bash
git clone https://github.com/someuser/somerepo.git
```

---

### View Remote URLs

```bash
git remote -v
```
